# FinCal Innovation – Goal Based Investment Calculator

An interactive **Goal-Based Investment Calculator** built for the **FinCal Innovation Hackathon**, conducted by the **Indian Institute of Technology (Banaras Hindu University) – IIT-BHU** as part of **TECHNEZ'26**, and co-sponsored by **HDFC Mutual Fund**.

This project focuses on **Investor Education and Awareness** by helping users understand how to plan investments for future financial goals using transparent financial assumptions.

## 💡 Problem Statement

Many financial calculators available online are **complex, non-interactive, and difficult for beginners to understand**.
The goal of this project is to build a **simple, intuitive, and educational financial calculator** that improves financial awareness and helps users plan investments effectively.

---

## 🚀 Solution

This application provides a **Goal-Based Investment Calculator** that helps users estimate the **monthly investment required to achieve a financial goal** while accounting for inflation and expected investment returns.

The application provides:

• Clear and transparent financial assumptions
• Interactive investment calculations
• Real-time financial visualizations
• Educational explanations of financial concepts

The tool is designed strictly for **illustrative and educational purposes**.

## 🧮 Financial Calculation Logic

### Step 1 – Inflate Goal Value

Future Goal Value:

FV = Present Cost × (1 + Inflation Rate)^Years

---

### Step 2 – Calculate Required Monthly SIP

Monthly rate:

r = Annual Return ÷ 12

Total months:

n = Years × 12

Required Monthly SIP:

Required SIP = FV × r ÷ [((1 + r)^n − 1) × (1 + r)]

Where:

FV = Inflated Goal Value
r = Monthly Return Rate
n = Total Number of Months

All calculations follow **standard industry financial formulas** used for educational illustration.
